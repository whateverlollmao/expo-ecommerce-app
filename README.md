# LUXE - E-Commerce App

A mobile e-commerce app built with React Native and Expo.

## Features

- Product catalog with categories
- Shopping cart and checkout
- Wishlist
- Multi-language (EN/RU/KK)
- Multi-currency support
- Dark/Light theme
- Order history
- Saved addresses
- Promo codes

## Setup

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go.

## Configuration

Edit `config/config.js` to add your API keys and backend URL.

## Project Structure

```
app/           - Screens (Expo Router)
components/    - UI components
context/       - State management
config/        - API config & settings
data/          - Product data
```
